# Weather App

Live weather app that shows current conditions and 5-day forecast for any city.

**Live Demo:** https://ShehreenFarooq.github.io/Weather/

### Features
- Search weather by city
- Shows temperature, humidity, wind, pressure
- 5-day forecast
- Works on mobile and desktop

### Built With
HTML, CSS, JavaScript, OpenWeatherMap API

### How to Run
1. Clone repo: `git clone https://github.com/ShehreenFarooq/Weather.git`
2. Open `index.html` in browser
